Project 2 Documentation

The purpose of this document is to make a game finder, where people can search for free games using an API with more than 400 free games. Also I used a lot of blue in the project to simulate Steam, which is another widely used game program.

The audience is mostly for people that are looking to download a game, but don't have the money for it. It's also for those interested in finding almost any type of game. Although I won't be able to put every single filter that can be put on the genres, the actual website has almost 20+ genres to look through. So I opted to only add about half of them so the selection list wasn't so long.
